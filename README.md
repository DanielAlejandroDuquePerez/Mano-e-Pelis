## 📋 Descripción
Aplicación web para buscar películas, ver sinopsis, calificaciones y pósters, y guardar tus favoritas. Consume datos reales de la API de TMDB en tiempo real.

## ✨ Funcionalidades
- 🔥 Películas populares del momento
- 🔍 Búsqueda en tiempo real por nombre
- 📄 Detalle completo de cada película
- ❤️ Lista personal de favoritos

## 🛠️ Tecnologías
| Tecnología | Uso |
|------------|-----|
| Python 3.13 | Lenguaje principal |
| Streamlit | Framework web |
| Requests | Consumo de API REST |
| TMDB API | Datos de películas |

## 🚀 Correr localmente
```bash
git clone https://github.com/DanielAlejandroDuquePerez/Mano-e-Pelis.git
cd Mano-e-Pelis
pip install -r requirements.txt
streamlit run app.py
> ⚠️ Necesitas crear `.streamlit/secrets.toml` con tu API key:
> `tmdb_api_key = "tu_api_key_aqui"`
